# Manhwa-Bookmark-Manager
Creates a bookmark manager by taking bookmark links from your browser and web scrapes the links for the name of the manhwa, current chapter, latest chapter, find how many chapters are available and then sorts it based on how many chapters are available from highest to lowest

Currently Supports only Asura Scans

In setup.py edit the path variable to the path of your browser's bookmark data.
For the program to find the path to your manhwas make a folder named "New folder" and name the folder inside "Manhwa" then put all your manhwa links inside.
This is how I organized my bookmarks, if you wish to edit your path feel free to do so.

Setup.py will automatically only serach for Asura Scans and edit your links if they are outdated, in the future I will improve upon this feature and add more scan sites

You can test load.py with the save.txt attached all you need to do is run it and you can have a preview of how your bookmark manager should look like
